Vector Space
Definition
A vector space is a set V of vectors, adorned with an additional feature of allowing two operations, namely, vector addition (+) and scalar multiplication (⋅), satisfying the following vector axioms for vectors u,v,w ∈ V and scalars a,b ∈ F, (where F = ℝ, ℂ, etc.) :
  • Commutativity of vector addition: u + v = v + u ,
  • Associativity of vector addition: u + (v + w) = (u + v) + w ,
  • Identity for vector addition: ∃0 ∈ V s.t. 0 + v = v = v + 0, ∀v ∈ V,
  • Inverse for vector addition: ∃ (−v) ∈ V s.t. (−v) + v = 0 = v + (−v),
  • Closure under both the operations: a ⋅ v ∈ V and v + w ∈ V ,
  • Distributivity over vector addition: a ⋅ (v + w) = a ⋅ v + a ⋅ w ,
  • Distributivity over scalar addition: (a + b) ⋅ v = a ⋅ v + b ⋅ v ,
  • Associativity of scalar multiplication: (ab) ⋅ v = a ⋅ (b ⋅ v) ,
  • Identity for scalar multiplication: ∃ 1 ∈ F such that 1 ⋅ v = v .

Examples of a vector space

  • ℝ : = ;
  • ℂ ;
  • ℝn : = , or , , etc. ;
  • ℝm×n : = ;
  • Function space: The set of functions {f ∣f : VW} , where V and W are vector spaces (as defined) and the operations (vector addition and scalar multiplication) are defined pointwise ;
  • Dual space, V∗ : = {g∣g : VF } , g is called a functional ;
  • Cn[a,b] : The set of all real-valued n times continuously differentiable functions defined on the interval [a,b] .
Linear Subspace
Definition
A linear subspace W is a non-empty subset of vector space V, adorned with the same operations, vector addition (+) and scalar multiplication (⋅), which satisfy the same vector axioms, i.e. in addition to the axioms for V, for vectors u,v ∈ W (⊆ V) and scalars a,b ∈ F, (where F = ℝ, ℂ, etc.) :
  • a ⋅ u + b ⋅ v ∈ W .

Examples:

  • Set of all vectors with second component 0, forms a subspace of ℝ3,
  • Set of all real symmetric n × n matrices forms a subspace of ℝn×n,
  • Cr[a,b] is a subspace of Cs[a,b] for all r > s,
  • Null space (or kernel) of a matrix Am×n : Solution space x ∈ ℝn of the homogenous system Ax = 0 is a subspace of ℝn.
Inner Product Space
Definition
A vector space V is called an inner product space if it admits a map ⟨⋅,⋅⟩ : V × V → F, called an inner product, satisfying the following axioms for vectors u,v,w ∈ V and scalars a,b ∈ F, (where F = ℝ, ℂ, etc.) :
  • Linearity in the first argument: ⟨a ⋅ u + b ⋅ v,w⟩ = a⟨u,w⟩ + b⟨v,w⟩ ,
  • Conjugate symmetry or Hermitian symmetry: ⟨u,v⟩ = ⟨v,u⟩ ,
  • Positive definiteness: ⟨v,v⟩ > 0, if v≠0 .

Alternatively, an inner product can be written with linearity in the second argument and conjugate linearity (antilinearity) in the first argument, as is the case with Dirac’s bra-ket notation, where ⟨v,w⟩ = v⟩. Direct Sum of matrices

Definition
A group, denoted as (G,⋆), is a set G of elements, which admits a binary operation ⋆ (called the group operation), satisfying the following group axioms for elements a,b,c ∈ G :
  • Closure: a ⋆ b ∈ G ,
  • Associativity: a ⋆ (b ⋆ c) = (a ⋆ b) ⋆ c ,
  • Identity: ∃ e ∈ G such that e ⋆ a = a = a ⋆ e, ∀ a ∈ G ,
  • Inverse: ∃ a−1 ∈ G such that a−1 ⋆ a = e = a ⋆ a−1 .

For multiplicative groups, the group operation (multiplication, ⋅) is implied by mere juxtaposition of elements, i.e. a ⋅ b ≡ ab. Examples of a group

  • (ℤ,+): set of integers under addition,
  • (ℝ,+): set of real numbers under addition,
  • (ℚ,+): set of rational numbers under addition,
  • (ℝ∖{0},×): set of non-zero real numbers under multiplication,
  • (ℚ∖{0},×): set of non-zero rational numbers under multiplication,
  • (ℝ+,×): set of positive real numbers under multiplication,
  • (ℚ+,×): set of positive rational numbers under multiplication,
  • (V,+), where set V forms a vector space and + is vector addition,
  • (kℤ,+): set of integer multiples of k ∈ ℤ under addition,
  • GL(n,F): set of n × n invertible matrices under matrix multiplication,
  • GL(V): set of all bijective linear transformations VV under function composition ∘,
  • U(n): set of n × n unitary matrices under matrix multiplication,
  • U(V): set of all unitary transformations VV (i.e. inner product preserving bijections) under function composition ∘ .
Subgroup
Definition
(H,⋆) is called a subgroup if H is a subset of set G that forms a group (G,⋆), such that H too forms a group under the same group operation ⋆.

A subgroup-group relation is denoted by the symbol ≤ .

Examples:

  • (ℤ,+) is a subgroup of (ℚ,+) ≤ (ℝ,+) ≤ (ℂ,+),
  • (ℚ+,×) is a subgroup of (ℝ+,×),
  • A linear subspace of a vector space V is a subgroup of (V,+),
  • SL(n,F): subgroup of GL(n,F), of matrices with unit determinant,
  • U(n): subgroup of GL(n, ℂ),
  • SU(n): subgroup of U(n) and SL(n, ℂ), of matrices with det = 1,
  • O(n): set of n × n orthogonal matrices forms a subgroup of GL(n, ℝ),
  • SO(n): subgroup of O(n) and SL(n, ℝ), of matrices with det = 1.

Linear Span and Basis
Definition
Given a vector space V and a set of vectors S : = , the linear span of set S, denoted span(S), is the set of all finite linear combinations of the elements of S, i.e. for vectors vi ∈ V and scalars ai ∈ F (where F = ℝ, ℂ, etc.), span(S) = .
The linear span of a set of vectors is therefore a vector space (particularly, a linear subspace W of V) and S ⊆ W.
Lemma
span(S) is the smallest subspace of V that contains the spanning set S.
V is said to have been spanned by S and S is called a generating set of V.
Definition
Smallest possible generating set (or a minimal spanning set) {ei} is called a basis, i.e. set {ei} is a maximal linearly independent set of basis vectors.
Generating Set of a group and Cyclic Group
Definition
A generating set of a group is a subset S of group G such that each element of G can be expressed as some combination (under the group operation ⋆) of finitely many elements of S and their inverses.

G is said to be generated by S and the elements of S are called generators.

To signify the same, G is denoted as ⟨S⟩.

By convention, ⟨∅⟩ is taken to be the trivial group ({e},⋆).

For a finite generating set S, G is said to have been finitely generated.

Definition
A cyclic group C is a group generated by a single element g. Therefore, ∀ k ∈ ℤ, ∃ c ∈ C such that c = gk.

Normal Subgroup

Definition
Two elements a,b ∈ G are similar (or conjugate to each other) if there exists g ∈ G, such that a = gbg−1 and the relation is called a similarity transformation (or conjugacy).

Definition
A subgroup gHg−1 ≤ G is called a conjugate subgroup of the subgroup H ≤ G, defined for a fixed g ∈ G as gHg−1 : = {ghg−1∣h ∈ H}.

Definition
A subgroup N ≤ G is called a normal subgroup (also called invariant or self-conjugate subgroup) of G iff ∀ g ∈ G, gNg−1 = N and is denoted as N ◃ G.

Definition
A simple group G is a group whose only normal subgroups are {e} and G.

Conjugacy Class and Center of a group

Definition
In a group G, the conjugacy class of an element x ∈ G is defined as clG(x) : = {gxg−1∣g ∈ G}, i.e. the set of all elements in G, which are conjugate to x.

Definition
The center of a group G is the set Z(G) of all the elements that commute with every element of G, i.e. Z(G) = {z ∈ G∣zgz−1 = g ∀ g ∈ G}.

Equivalently, the center is the set of elements, which are the only elements in their respective conjugacy classes, i.e. x ∈ Z(G) ⇔ clG(x) = {x}.

Theorem
The center of a group G is its normal subgroup, Z(G) ◃ G.

Abelian Group

Definition
An abelian group (G,⋆) is a group in which the elements of set G commute under the binary operation ⋆ , i.e. in addition to the group axioms, for elements a,b ∈ G, following is satisfied :
  • Commutativity: a ⋆ b = b ⋆ a .

For an abelian group G, Z(G) = G.

Definition
Direct sum of abelian groups (G,⋆) and (H,∙) is the same as their direct product, i.e. G ⊕ H = (G × H,∗), such that ∗ is defined component-wise:
  • (g1,h1) ∗ (g2,h2) = (g1 ⋆ g2,h1 ∙ h2), ∀ g1,g2 ∈ G, h1,h2 ∈ H.

Definition
Order of an element g ∈ G is the smallest positive integer m such that gm ≡ g ⋆ g ⋆ g ⋆ …(m times) = e, where e is the identity element of G.

If no such m exists the order of the element is said to be infinite.

Definition
Order of a group G, denoted by |G|, is the number of elements in its set.

If |G| is finite, G is called a finite group.

Definition
Conjugacy class order is the number of elements in a conjugacy class.

Definition
Class number of a group is the number of distinct conjugacy classes.

Lagrange’s Theorem

Theorem
The order of every subgroup of a finite group G is a factor of |G|.

Corollary
The order of every element of a finite group G is a factor of |G|.

Corollary
∀ g ∈ G, g|G| = e.

Corollary
Every group G of prime order (i.e. |G| being prime) is cyclic, hence simple.

Corollary
The order of every conjugacy class of a finite group G is a factor of |G|.

Sylow Theorems

Theorem
For a prime number p and a finite group G, if |G| = pnq (p doesn’t divide q), there exists a subgroup of G, of order pk for each k ∈{1,…,n}.

Subgroup of order pn is known as a Sylow p-subgroup of G.

There exists a subgroup of G, of order q, an example of a Hall subgroup.

If q = 1, G is called a p-group and ∀g ∈ G, order |g| is some power of p.

Corollary (Cauchy’s theorem)
For a finite group G, if p is a prime factor of |G|, there exists an element (and thus a cyclic subgroup generated by this element) of order p in G.

Theorem
For a prime number p, all Sylow p-subgroups are conjugate to each other.

Theorem
Number np of Sylow p-subgroups of G divides |G|, while p divides (np − 1).

Cayley Table A Cayley table is a succinct representation of the group structure.
Definition
A Cayley table is a square table used to describe the structure of a finite group by arranging the group elements following the group operation ⋆.

Convention: An element (say a) that labels a row appears before an element (say b) that labels a column, in the corresponding entry of the table, i.e. as the product a ⋆ b (instead of an alternate convention of b ⋆ a).

Theorem
Each row and each column in a Cayley table lists every group element exactly once. This is known as the rearrangement theorem.

Caution: A Cayley table doesn’t admit associativity as an axiom for its construction. Therefore, any structure (formed by the elements of a set under the defined operations) obtained through the application of the rearrangement theorem must still be explicitly checked for associativity before accepting it as a group structure. Definition
With respect to elements g ∈ G, the left cosets of a subgroup (H,⋆) of a group (G,⋆) are given by gH : = {g ⋆ h∣h ∈ H}, and the right cosets of H are given by Hg : = {h ⋆ g∣h ∈ H}.

Left (or right) cosets of H ≤ G are disjoint subsets of G.

Each (left or right) coset of H ≤ G in G has |H| number of elements.

As seen already, N ◃ G ⇔ gN = Ng, ∀ g ∈ G.

Definition
The index of a subgroup H in a group G is the number of left (or right) cosets of H in G and is denoted by |G : H|.

Equivalence Class and Quotient Set

Definition
The equivalence class [a] of an element a in set S, with respect to an equivalence relation ∼, is the set of elements in S that are equivalent to a, [a] : = {x ∈ S∣a ∼ x}, i.e. the binary relation ∼ satisfies ∀ a,b,c ∈ S:
  • Reflexivity: a ∼ a ,
  • Symmetry: a ∼ b ⇔ b ∼ a ,
  • Transitivity: a ∼ b and b ∼ c a ∼ c .

Each element of a set belongs to exactly one of its equivalence classes.

Definition
The set S∕∼ of all equivalence classes of a set S is called a partition of S by ∼ or a quotient set of S by ∼ .

If a set S has an additional structure (e.g. that of a vector space or a group) and the equivalence relation ∼ on S is compatible with that structure, its quotient set S∕∼ inherits that structure. Quotient Space and Quotient Group

Definition
For a vector space V and its linear subspace W, the quotient space V∕W is the quotient set V∕∼, where the equivalence relation ∼ on V is defined as v1 ∼ v2 ⇔ v1 − v2 ∈ W, i.e. [v] = {v + w∣w ∈ W}. This set forms a vector space with the operations defined for vectors v1,v2 ∈ V and scalars a ∈ F, (where F = ℝ, ℂ, etc.), as:
  • a ⋅ [v1] = [a ⋅ v1] ,
  • [v1] + [v2] = [v1 + v2] .

Definition
For a group (G,⋆) and its normal subgroup N, the quotient group G∕N is the set of all left (or right) cosets of N in G, i.e. G∕N : = {gN∣g ∈ G}. This set forms a group with the group operation ∙ defined ∀g1,g2 ∈ G as:
  • (g1N) ∙ (g2N) = (g1 ⋆ g2)N .

Structure preserving maps
Definition
A group homomorphism is a map f : GH, i.e. from group (G,⋆) to (H,∙), which preserves the group structure, viz. the identity and the group operation, by satisfying f (g1 ⋆ g2) = f (g1) ∙ f (g2), ∀g1,g2 ∈ G.
As a result,
  • f (eG) = eH, where eG and eH are identity in G and H, respectively.
  • f (g1−1) = (f (g1))−1, i.e. inverse in G maps to inverse of image, in H.

Thus, the map f is said to be compatible with the group structure of G.

Definition
The kernel of f : GH is the set of elements in G that are mapped to the identity element (eH) of H, i.e. ker(f ) ≡{g∣f (g) = eH, g ∈ G}.

Definition
The image of f : GH is the set of elements in H to which elements of G are mapped, i.e. im(f ) ≡ f (G) ≡{f (g)∣g ∈ G}.

Structure preserving maps (contd.)

A group homomorphism f : GH is said to be trivial if ker(f ) = G.

Definition
A monomorphism is a group homomorphism that is injective (one-one).

Definition
An epimorphism is a group homomorphism that is surjective (onto).

Definition
An isomorphism is a group homomorphism that is bijective (invertible).

Definition
An endomorphism is a group homomorphism of a group to itself.

Definition
An automorphism is a bijective (isomorphic) endomorphism of a group.

Group Action

Definition
A group action ϕ(g,x) of a group (G,⋆) on a set X is a bijective (or invertible) map ϕ : G × XX, satisfying the following axioms ∀x ∈ X :
  • Identity: ϕ(e,x) = x, where e is identity in G,
  • Compatibility: ϕ(g,ϕ(h,x)) = ϕ(g ⋆ h,x), ∀ g,h ∈ G.

Set X, together with an action of group G, is called a G-set.

The above definition is the case of a left group action ϕL : G × XX, usually adopted as a convention. Alternatively, one can define a right group action ϕR : X × GX, satisfying analogous axioms.

Definition
An inner automorphism is the group action of a group G on itself, given by the map ϕg : G × GG for a fixed g ∈ G, defined as ϕg(x) : = g−1xg.

Orbits under group action

Definition
The orbit of an element x ∈ X under action ϕ(g,x) of a group (G,⋆) is the set of elements in X to which x gets mapped, under the group action, i.e. G ⋅ x : = {ϕ(g,x)∣g ∈ G}.

Definition
Length of the orbit G ⋅ x of an element x ∈ X under the action of a group G is the number of elements in that orbit and is denoted as |G ⋅ x|.

Definition
The quotient of action (denoted as X∕G) of a group G on set X is the set of all orbits of elements in X under the action ϕ(g,x) of G, i.e. it is the quotient set X∕∼, where the equivalence relation ∼ on X is defined as x ∼ y ⇔ ∃ g ∈ G such that y = ϕ(g,x).

Any additional structure on set X is inherited by the quotient of action. Exercise Verify that ∼ forms a partition of X. Stabilizer

Definition
For x ∈ X and g ∈ G, x is said to be a fixed point of g if it is invariant under the action ϕ of group G on set X, i.e. ϕ(g,x) = x.

Definition
The stabilizer Gx of a group G with respect to an x ∈ X is the set of elements in G with x as their fixed point, i.e. Gx : = {g ∈ G∣ϕ(g,x) = x}.

Definition

The isomorphism theorems can be proven using the fundamental theorem on homomorphisms, which states:

Theorem
Given an arbitrary group homomorphism f : GH and an epimorphism ϕ : GG∕N, defined by ϕ : ggN for N ◃ G such that N ⊆ ker(f ), there exists a unique homomorphism α : G∕NH with α(gN) = f (g).

Matrix Transformations
Lemma
Any linear transformation between finite-dimensional vector spaces can be represented by a matrix.
Conversely, every matrix transformation is a linear transformation.
Proposition
For every matrix Am×n with entries in F (where F = ℝ, ℂ, etc.), there exists the linear transformation T : FnFm, given by T(v) = Av.

Diagonalizability of matrices
Definition
A square matrix is called diagonalizable if it is similar (associated through a similarity transformation) to a diagonal matrix.

Equivalently, diagonalizability ⇔ linear independence of eigenvectors.

The canonical approach of diagonalization is called eigendecomposition, in which the diagonal entries of the associated diagonal matrix are the eigenvalues of the original matrix and the columns of the diagonalizing matrix are the eigenvectors of the original matrix.

Lemma
If a matrix A represents the linear transformation T : VV of a vector space V, A is diagonalizable iff a basis of V comprises eigenvectors of A.

Proposition
For a finite order matrix transformation T of a finite-dimensional vector space V, (i.e. Tn = 𝕀, n ∈ ℕ), a basis of V comprises eigenvectors of T.

Spectral Theorems for matrices

Spectral theorem for Hermitian matrices:

Theorem
For a(n) Hermitian matrix:
  • all eigenvalues are real,
  • eigenvectors corresponding to distinct eigenvalues are orthogonal,
  • there exists an orthogonal basis of the vector space consisting of those eigenvectors.

Corollary
Hermitian matrices are unitarily diagonalizable, i.e. for every Hermitian matrix H, there exists a unitary matrix U such that U−1HU is diagonal.

The diagonal entries of U−1HU are the (real) eigenvalues of H and the columns of U are the (orthogonal) eigenvectors of H. Spectral Theorems for matrices (contd.)

Spectral theorem for unitary matrices:

Theorem
For a unitary matrix:
  • all (complex) eigenvalues have absolute value 1,
  • eigenvectors corresponding to distinct eigenvalues are orthogonal,
  • there exists an orthogonal basis of the vector space consisting of those eigenvectors.

Corollary
Unitary matrices are unitarily diagonalizable, i.e. for every unitary matrix U, there exists a unitary matrix U such that U−1UU is diagonal.

The diagonal entries of U−1UU are the eigenvalues (with unit modulus) of U and the columns of U are the (orthogonal) eigenvectors of U. Linear Representation

Abstract group elements are described in terms of bijective (invertible) linear transformations (automorphisms) of vector spaces.

Definition (1)
A linear representation ϕ of a group (G,⋆) is the group action of G on a vector space V, i.e. ϕ : G × VV, given by a map ϕ(g) : vϕ(g,v), satisfying the axioms for group action and linearity in the second argument.

Dimension of vector space V is called the dimension of the representation.

Definition (2)
A linear representation ρ of a group (G,⋆) is the group homomorphism ρ : GGL(V) from G to the general linear group GL(V) on a vector space V, given by a map ρ(g) : vρ(g,v), satisfying the axioms for group homomorphism and linearity in the second argument.

Groups elements are usually represented by invertible matrices (sometimes called representatives) and the group operation by matrix multiplication. Linear Representation (contd.)

Definition
Kernel of a representation ρ : GGL(V) is the set of preimage(s) of the identity transformation in V (idV ), or ker(ρ) = {g∣ρ(g) = idV , g ∈ G}.

Definition
A faithful representation is a representation that is a monomorphism.

Kernel of a faithful representation is trivial and no two elements of the group have the same representative.

Definition
Representations ρ : GGL(V) and σ : GGL(W) are isomorphic if there exists an isomorphism α : VW s.t. ∀ g ∈ G, α ∘ ρ(g) = σ(g) ∘ α.

The map α is said to be equivariant when representations are isomorphic.

Definition
A unitary representation is a group homomorphism ρ : GU(V).

Direct Sum of representations

Definition (1)
Direct sum of representations ρ1 : G1GL(V1) and ρ2 : G2GL(V2) is a group homomorphism ρ1 ⊕ ρ2 : G1 × G2GL(V1 ⊕ V2) such that (ρ1 ⊕ ρ2)((g1,g2),v1 ⊕ v2) = ρ1(g1,v1) ⊕ ρ2(g2,v2), ∀ g1 ∈ G1,g2 ∈ G2, v1 ∈ V1,v2 ∈ V2.

Definition (2)
Direct sum of representations (V1,ρ1) and (V2,ρ2) of groups G1 and G2, respectively, is a linear representation (V1 ⊕ V2,ρ) of the group G1 × G2 whose action α((g1,g2),v1 ⊕ v2) on V1 ⊕ V2 is defined component-wise:
  • α((g1,g2),v1 ⊕ v2) = ϕ1(g1,v1) ⊕ ϕ2(g2,v2), ∀ g1 ∈ G1, g2 ∈ G2, v1 ∈ V1, v2 ∈ V2, ϕ1 : G1 × V1V1 and ϕ2 : G2 × V2V2 .

Direct product of representations ρ1:G1GL(V1) and ρ2:G2GL(V2) is a group homomorphism ρ1 ⊗ ρ2 : G1 × G2GL(V1 ⊗ V2) such that (ρ1 ⊗ ρ2)((g1,g2),v1 ⊗ v2) = ρ1(g1,v1) ⊗ ρ2(g2,v2), ∀ g1 ∈ G1,g2 ∈ G2, v1 ∈ V1,v2 ∈ V2.

Direct product of representations (V1,ρ1) and (V2,ρ2) of groups G1 and G2, respectively, is a linear representation (V1 ⊗ V2,ρ) of the group G1×G2 whose action α((g1,g2),v1 ⊗ v2) on V1 ⊗ V2 is defined component-wise:
  • α((g1,g2),v1 ⊗ v2) = ϕ1(g1,v1) ⊗ ϕ2(g2,v2), ∀ g1 ∈ G1, g2 ∈ G2, v1 ∈ V1, v2 ∈ V2, ϕ1 : G1 × V1V1 and ϕ2 : G2 × V2V2 .

Definition
A subrepresentation σ : GGL(W) of a representation ρ : GGL(V) is a map that is preserved by group action, i.e. ∀ w ∈ W and g ∈ G, σ(g,w) ∈ W, where W is a linear subspace of V.

Clearly σ is also a representation of the group G.

Indicatively, the subspace W itself is referred to as the subrepresentation.

A trivial (sub)representation VG(g) : VV is the map VG(g) : vv.

Definition
A non-trivial representation (V,ρ) is called an irreducible representation (or irrep) if it has exactly two subrepresentations, viz. ({0},σ) and (V,ρ).

Definition
A reducible representation is a non-trivial representation with more than two subrepresentations.

Subrepresentation and Reducibility (contd.)

Theorem
Given two linear representations (V,ρV ) and (W,ρW ) of a group G and a homomorphism α : VW, such that ∀ g ∈ G, α ∘ ρV (g) = ρW (g) ∘ α, ker(α) is a subrepresentation of V and im(α) is a subrepresentation of W.

Definition
A representation V is said to be decomposable if it is a direct sum of its subrepresentations (≠{0},V). Else, it is said to be indecomposable.

In terms of matrix representations, a representation is decomposable if all representative matrices can be block diagonalized upon a similarity transformation by the same invertible matrix, else it is indecomposable.

Each block is a subrepresentation of the matrix representation.

Definition
Matrix representations which are associated with each other through a similarity transformation are called equivalent representations.

More on irreducible representations

Theorem
Irreducible representations are always indecomposable.

The converse holds for finite-dimensional complex representations of finite groups, but may not be true in general.

In terms of matrix representations, an irrep is one for which there exists no unitary transformation (similarity transformation through a unitary matrix) that will block diagonalize all the representative matrices simultaneously (i.e. using the same diagonalizing matrix).

Theorem

If (V,ρV ) is a representation of a finite group with a subrepresentation (W,ρW ), there exists a complementary subrepresentation (W′,ρW′) of (V,ρV ) such that ρV = ρW ⊕ ρW′.

Schur’s Lemma
Lemma
For two irreducible representations (V,ρV ) and (W,ρW ) of a group G, the homomorphism α : VW, such that ∀ g ∈ G, α ∘ ρV (g) = ρW (g) ∘ α, is either identically zero or an isomorphism (invertible).
Clearly, if the two representations are not isomorphic, for the map α to remain equivariant, it must be identically zero.
Corollary
If (V,ρV ) and (W,ρW ) are isomorphic, the space of all homomorphisms α : VW is one-dimensional.
Corollary
For V = W and ρV = ρW , any endomorphism α : VV is a scalar multiple of the identity transformation in V (idV ), i.e. α = λidV , λ ∈ ℂ.
Schur’s Lemma (contd.)
Corollary
If a finite group G has a faithful irreducible representation, then its center Z(G) is a cyclic subgroup of G, isomorphic to that of (ℂ∖{0},×).

Corollary
All irreducible representations of finite abelian groups are one-dimensional.

Corollary
A finite abelian group G has exactly |G| irreducible representations.

Character

Definition
The character of a linear representation ρ : GGL(V) (where V is a vector space over F = ℝ, ℂ, etc.) is the function χρ : GF given by χρ(g) = Tr(ρ(g)), ∀g ∈ G.

For a matrix representation of a group, the character of the representation for any given element is the trace of the representative matrix for that element in that representation.

A character χρ is called irreducible if ρ is an irreducible representation.

Definition
Kernel of a character is defined as ker(χρ) : = {g ∈ G∣χρ(g) = χρ(e)}, where e is the identity in G.

Clearly, ker(χρ) = ker(ρ), hence, a normal subgroup of G.

A character χρ is faithful if ker(χρ) is trivial.

im(χρ) does not form a group, in general. Character (contd.)

Theorem
Character of a direct sum (product) representation of representations ρ and σ is the sum (product) of the characters χρ and χσ of the respective representations, i.e. χρ⊕σ = χρ + χσ and χρ⊗σ = χρ ⋅ χσ.

Theorem
For a representation ρ of a group G, χρ∗(g) = χρ(g−1) = χρ(g), ∀ g ∈ G.

Dimension of a representation ρ is called the degree of the character χρ.

A character with unit degree is called a linear character.

Theorem
The number of complex irreducible characters of a group G is equal to the number of conjugacy classes of G, and their degrees divide |G|, |G : Z(G)|.

Class Function

Definition
A function on a group, which is invariant under an inner automorphism on that group, is called a class function. Equivalently, it is one which maps all the members of a conjugacy class to the same element in its image.

Character of a representation of a group G is always a class function on G.

Proposition
The set of class functions f i : GF on a group G forms a vector space over F, whose dimension equals the number of conjugacy classes in G.

Proposition
The complete set of complex irreducible characters χi of a group G forms an orthonormal set with respect to the inner product of complex class functions on G, i.e. χi⟩ = ⟨χi,χj⟩ = δij.

Theorem
The complete set of complex irreducible characters χi of a group G spans the inner product space L2(G) of complex class functions on G, hence forming an orthonormal basis of the same.
